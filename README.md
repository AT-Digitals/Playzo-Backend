## Quick Start

Setup environment variables:

- Copy `.env.example` file and rename as `.env`
- Update the `.env` file constants

Install dependencies:

```bash
$ npm install
```

Start the server:

```bash
$ npm start
```

View the website at: http://localhost:8000
